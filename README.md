# 鹏城分类垃圾大挑战·小游戏

## 游戏介绍

<strike>本游戏旨在提高大家对生活垃圾分类的认识，寓教于乐。游戏分为简单模式和困难模式，让您在轻松愉快的氛围中掌握垃圾分类知识。</strike>
如果你觉得难，它确实是难的。
只是，我想的是：在面对没有分类的垃圾，点下按钮就能把垃圾放到正确的垃圾桶里，也并不是一件很难的事情啦！
因为，当我想到：如果是闻着各种没有正确分类的垃圾，在垃圾桶里分类垃圾，这肯定是更难的事。
而最难的，可能是十几亿人都分类垃圾的意愿吧！

## 游戏模式

*   **简单模式:** 垃圾掉落速度较慢，垃圾带有类别颜色提示。适合新手入门。
*   **困难模式:** 垃圾掉落速度较快，没有颜色提示，更具挑战性，适合有一定垃圾分类基础的玩家。

因个人原因未能实现的功能：
1.排行榜，并且100分的玩家都并列第一！
2.困难100分，游戏就结束了，只有自动产生垃圾，自动分类垃圾，不断加分的动画！
————我想，未来，一定会有，了不起的人工智能，帮助大家分类垃圾！

## 游戏说明

1.  **分类说明:** 点击问号按钮，可以查看详细的生活垃圾分类说明。
2.  **时间限制:** 每局游戏时长为 60 秒。
3.  **计分规则:**
    *   正确分类垃圾：+5 分
    *   错误分类垃圾：-3 分

## 交互方式

游戏下方有三个按钮：

*   **[交换左2]:** 交换最左边两个相邻的垃圾桶的位置。
*   **[交换中2]:** 交换中间两个相邻的垃圾桶的位置。
*   **[交换右2]:** 交换最右边两个相邻的垃圾桶的位置。

在垃圾掉落之前，通过点击按钮调整垃圾桶的位置，确保正确的垃圾桶位于垃圾下方，即可成功分类垃圾。
